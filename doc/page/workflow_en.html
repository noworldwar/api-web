Work flow

  • Launch game

    • note：1
      Game URL Format：
      https://{Game Platform URL}?token={Token generated by WE platform}&gameid={Game ID}&lang={Language}&playmode={Play Mode}

      Real Mode Example：
      https://www.gameserver.com?token=abcdef123456&gameid=1&lang=en&playmode=real

      Demo Mode Example：
      https://www.gameserver.com?gameid=1&lang=en&playmode=demo


  • Operation in-game

    • note：2
      If the token has expired, please refer to the following steps:
      1. WE platform will confirm the order is correct and then give back the payout to the player whilst return "tokenExpired=true"
      2. Request the game provider to sign out the player, and show "Token has expired, please log in again"


  • Cancel bet
